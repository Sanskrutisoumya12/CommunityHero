# CommunityHero

CommunityHero is a Django-based community issue reporting platform that enables citizens to report local civic problems such as broken street lights, potholes, garbage issues, and more.

## Features

- User Registration & Login
- OTP Verification
- Issue Reporting with Images
- AI-generated Issue Summary
- Authority Dashboard
- Leaderboard
- Notifications
- Profile Management

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
- Google Gemini API

## Installation

```bash
git clone https://github.com/Sanskrutisoumya12/CommunityHero.git

cd CommunityHero

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
